# Medário wordmarks

Transparent PNG variants:

- `medario-wordmark-on-light.png`: blue wordmark for light surfaces.
- `medario-wordmark-on-dark.png`: white wordmark for dark surfaces.
- `medario-wordmark-tagline-on-light.png`: blue wordmark and tagline for light surfaces.
- `medario-wordmark-tagline-on-dark.png`: white wordmark and tagline for dark surfaces.

The red leaf mark remains unchanged in every variant.
